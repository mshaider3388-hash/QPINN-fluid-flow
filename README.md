 QPINN Fluid Flow Analysis

This project presents a Physics-Informed Neural Network (QPINN) approach to analyze viscous fluid flow over a wedge. It combines principles of physics with deep learning to solve complex differential equations governing fluid dynamics.


  Project Overview

Traditional numerical methods for solving fluid flow problems can be computationally expensive. This project uses Physics-Informed Neural Networks (PINNs) to:

- Solve governing equations of viscous fluid flow
- Reduce dependency on large datasets
- Incorporate physical laws directly into the learning process


 Methodology

- Formulation of governing differential equations
- Implementation of neural networks
- Loss function based on physical constraints
- Training using optimization techniques

  
 Tools & Technologies

- Python
- PyTorch
- NumPy
- Matplotlib


 Features

- Data-efficient modeling using physics laws
- Continuous solution approximation
- Applicable to real-world fluid dynamics problems
